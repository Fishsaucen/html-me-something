In Search of the Perfect Keyboard

Qualities

Some of the qualities which I find desirable for the typing experience:

  • Layout
  • Switches
  • Feedback

Thermaltake Poseidon Z

Layout: Standard US ANSI. This layout is the standard for most keyboards produced today. While I find it generally acceptable to type with it has a few problems which I'd like to address:

  • Caps Lock location: I honestly can't imagine why this key is placed where it is. It's in prime real estate for a something that is seldomly used in my experience.
  • Escape key location: This boils down to my text editor of choice, VIM. For most people it's location isn't a issue but if you're familiar with VIM and it's normal/insert mode you'll see why this is a problem. When typing in VIM the user is constantly changing between normal/insert mode and the default key to exit insert mode is Escape which is a pain with this layout.

Switches: Cherry MX Brown. These switches are in the light range for mechanical keyboards(45cN) and are in my opinion decent for typists. They're light enough that you can type for extended periods of time without experiencing fatigue while also providing more resistance than traditional rubber dome keyboards which I find makes the typist more cognizant of each stroke thus preventing accidental double taps.

Feedback: About halfway through the key travel the user will notice a slight bump which indicates that a key has been pressed. This feature is great as it allows the user to avoid fully depressing the key.

Other notable features: Comes with a function key which allows the use of media keys and a special key to toggle whether or not the Meta key(Win/Command) can be used.

Sun Type 6, Unix

Layout: ANSI "Unix". Very similar to the ANSI US layout with some notable changes. The Caps Lock key is exchanged with the left control key(a feature which I love), the backspace key is located where the \ key is normally located and in its place are the \ and ~. The Escape key populates the traditional position of the ~ character which makes typing in VIM a much more pleasurable experience. Last but not least this keyboard comes with a compose key which allows typing special characters such as ®, £, or ¶ with relative ease.

Switches: Rubber dome. I'm personally not a fan of rubber dome switches but most users won't mind.

Feedback: Virtually no feedback is given until the key is fully depressed. Again, I'm not personally a fan of these but most users shouldn't mind.

Other notable features: Features two columns of five keys which can be assigned using a utility such as AutoHotKey.

Wyse-50

Layout: ANSI "Unix". Similar layout to the Sun Type 6 with a few notable exceptions. The most apparent difference in this keyboard's layout is the arrow key cluster at the bottom right of the keyboard left under the Enter key. The key cluster which usually contains Home, End, Del, etc. on full sized keyboards has also been merged with the main keys; consequently, keys such as shift, Enter and backspace have a reduced size.

Switches: Cherry MX Black. These switches fall into the heavy range for mechanical keyboards(60cN). Due to the increased force required for a key press these switches may result in fatigue.

Feedback: Unlike the Cherry MX Brown switches these do not provide a tactile bump anywhere during a key press.

Other notable features: Features stepped keys(tab, ctrl, caps, enter, backspace, shift) which can help typists avoid accidently pressing those those keys. More importantly the interface is a din-10 plug which seems to only be supported by certain Wyse terminals. Luckily users at geekhack have created custom firmware using the Atmega32U4 microcontroller which allows the use of this keyboard on modern PCs after some simple modifications are made.

HHKB Professional 2

Layout: ANSI "Unix" 60%. Reminiscent of the Sun Type 6 except it lacks dedicated arrow keys, the numpad, and dedicated function keys. Great for those looking for a Unix style keyboard with little desk space to spare. The lack of the aforementioned keys are made up for by providing them with a function overlay.

Switches: Topre(45cN). Users describe these switches as somewhat of a "refined rubber dome" with traits similar to mechanical switches.

Feedback: Provides a tactile bump near the top of the key before the key is activated. This can make it somewhat difficult for unaccustomed users to avoid bottoming out.

Other notable features: N/A
